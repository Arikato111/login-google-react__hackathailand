# React with firebase
Learn React with firebase from [HackaThailand](https://hackathailand.com/courses/basic-web-application-development-full-stack/)

## Get started

- Enter your `firebaseConfig` at `./src/firebase.js` -- $!important$ --
- `npm install`
- `npm start`

## Inside it

- antd
- firebase
- axios
- @emotion
